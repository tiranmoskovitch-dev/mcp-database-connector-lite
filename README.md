# MCP Database Connector Lite

Free, minimal SQLite MCP server. Connect your AI agent to a SQLite database in 30 seconds.

## Quick Start

```bash
pip install mcp
export MCP_DB_DATABASE=./my_data.db
python server.py
```

## Tools

| Tool | Description |
|------|-------------|
| `db_query` | Execute SQL queries |
| `db_list_tables` | List all tables |
| `db_describe_table` | Get table schema |
